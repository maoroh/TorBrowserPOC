# UnionRoutingPOC
Union Routing POC Implementation in Java
This project done for Computer Security Course , Open University.

## Execute Server

The server must listening before the client is up.
Use gradle for build executable jar.

```gradle
gradle serverJar
```


## Execute Client

Use gradle for build executable jar.

```gradle
gradle clientJar
```
